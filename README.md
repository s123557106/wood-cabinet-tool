# 台灣木工櫃體 3D 裁切計算工具 MVP v2.5

## 這版修正重點

### 五金勾選可以手動取消並收合

v2.4 的問題是：

- 只要選擇門片，系統每次重新計算都會自動勾回「估算鉸鍊」
- 只要勾選抽屜，系統每次重新計算都會自動勾回「估算滑軌」
- 導致使用者無法直覺地把鉸鍊或滑軌取消

v2.5 已修正：

- 第一次選擇門片時，自動勾選鉸鍊
- 使用者手動取消鉸鍊後，不會再被自動勾回
- 取消勾選後，鉸鍊設定會收合
- 鉸鍊成本不再計入總成本
- 第一次勾選抽屜時，自動勾選滑軌
- 使用者手動取消滑軌後，不會再被自動勾回
- 取消勾選後，滑軌設定會收合
- 滑軌成本不再計入總成本
