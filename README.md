# 台灣木工櫃體 3D 裁切計算工具 MVP v3.1

## 這版修正重點

### 1) 夾板與藝豐合板分開

v3.0 把市面公規夾板與藝豐廠家合板混在同一個「夾板」選單裡，導致右側選單很凌亂。

v3.1 已拆成兩個板材類型：

- 夾板：使用市面公規
- 藝豐合板：使用藝豐廠家規格

### 2) 夾板使用公規

夾板選單改為：

- 1分夾板｜0.27 cm
- 2分夾板｜0.36 cm
- 2分足夾板｜0.5 cm
- 7mm夾板｜0.7 cm
- 3分夾板｜0.9 cm
- 4分夾板｜1.2 cm
- 5分夾板｜1.5 cm
- 6分夾板｜1.8 cm
- 7分夾板｜2.1 cm
- 8分夾板｜2.4 cm

### 3) 新增「藝豐合板」

藝豐合板選單使用你提供的廠家規格，並統一用 cm 顯示：

- 藝豐合板 0.18 cm：3×6
- 藝豐合板 0.24 cm：3×6
- 藝豐合板 0.27 cm：3×7、4×8
- 藝豐合板 0.36 cm：3×6、3×7、4×8
- 藝豐合板 0.5 cm：3×6、3×7、4×8
- 藝豐合板 0.7 cm：3×6、3×7、4×8
- 藝豐合板 0.9 cm：3×6、3×7、4×8
- 藝豐合板 1.2 cm：3×6、3×7、4×8
- 藝豐合板 1.5 cm：3×6、3×7、4×8
- 藝豐合板 1.8 cm：3×6、3×7、4×8

### 4) 單位統一

選單顯示統一改成 cm，避免 mm / cm 混用。
