# 台灣木工櫃體 3D 裁切計算工具 MVP v2.9

## 這版新增重點

### 1) 箱體前板 / 後板可分別設定

箱體模式新增：

- 前板模式：不使用 / 內崁 / 外蓋
- 後板模式：不使用 / 內崁 / 外蓋
- 前後板外蓋量

前板和後板可以分開設定，例如：

- 前板不使用，後板內崁
- 前板外蓋，後板內崁
- 前後板都外蓋
- 前後板都不使用

### 2) 裁切尺寸會跟著模式變

內崁時：

- 前後板會扣左右板厚
- 若是有頂底板的箱體，也會扣頂底板厚度

外蓋時：

- 前後板會依外寬外高加上外蓋量

### 3) 3D 示意會同步改變

- 內崁前後板會顯示在箱體內側
- 外蓋前後板會顯示在箱體外側
- 淺盤 / 托盤的前後牆也支援內崁 / 外蓋

### 4) 自主優化

這版同時增加：

- 箱體形式切換時，自動套用建議前後板配置
- 箱體說明會顯示前板、後板目前模式
- 顯示內部可用深度估算
- 若前後板都不使用，會出現提醒
- 若內部深度不足，會出現提醒
