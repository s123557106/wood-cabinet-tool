# 台灣木工櫃體 3D 裁切計算工具 MVP v2.1

## 這版新增重點

### 櫃體 / 箱體模式切換

v2.1 新增功能頁切換：

- 櫃體模式
- 箱體模式

電腦版與手機版都可以直接切換。

## 櫃體模式

保留原本功能：

- 櫃體尺寸
- 背板
- 門片
- 分段門片
- 抽屜
- 五金
- 板材排版
- 成本估算
- 3D 示意

## 箱體模式

新增箱體專用計算：

- 前方開口箱
- 六面封閉箱
- 上方開口箱
- 淺盤 / 托盤
- 獨立上蓋
- 內底板

箱體模式會自動產生：

- 箱體左右側板
- 頂板 / 底板
- 前板 / 後板 / 背板
- 上蓋板
- 內底板

並同步計算：

- 裁切清單
- 板材排版圖
- 板材張數
- 利用率
- 成本
- 3D 示意

## 適合用途

- 木箱
- 道具箱
- 機關箱
- 收納箱
- 展示箱
- 運輸箱
- 托盤

## 更新 GitHub Pages

把 ZIP 解壓縮後，把裡面的檔案全部上傳覆蓋 GitHub repository 最外層：

- index.html
- style.css
- app.js
- manifest.json
- service-worker.js
- icon-192.png
- icon-512.png
- README.md

Commit changes 後等待 1–5 分鐘。

## 手機若沒更新

PWA 可能快取舊版。建議：

1. 重新整理網頁
2. 關掉瀏覽器重開
3. 刪除主畫面 App 後重新加入
