## 如何在手機使用

這個工具不能只靠 ZIP 檔直接在手機安裝成 App。

你需要先把整個資料夾上傳到一個 HTTPS 網站，例如：

- GitHub Pages
- Netlify
- Vercel
- Cloudflare Pages
- 自己的主機

上線後會得到一個網址，手機打開該網址即可使用。

## iPhone 加入主畫面

1. 使用 Safari 打開網站
2. 點分享按鈕
3. 選「加入主畫面」
4. 命名，例如「櫃體計算」
5. 之後桌面會出現 App 圖示

## Android 加入主畫面

1. 使用 Chrome 打開網站
2. 點右上角選單
3. 選「安裝應用程式」或「加到主畫面」
4. 之後桌面會出現 App 圖示

## 本機測試提醒

如果你直接用電腦打開 `index.html`：

- 功能可以正常使用
- 但 PWA 安裝與離線快取不會完整啟動
- 因為 Service Worker 需要 HTTP/HTTPS 環境
