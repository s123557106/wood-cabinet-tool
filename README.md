# 台灣木工櫃體 3D 裁切計算工具 MVP v3.2

## 這版新增重點

### 1) 新增內尺寸輸入

尺寸輸入方式可切換：

- 用外尺寸輸入
- 用內尺寸輸入

使用內尺寸輸入時，可以直接填：

- 內寬
- 內高
- 內深

系統會自動反推：

- 外寬
- 外高
- 外深

### 2) 內深會考慮內崁門與內嵌背板

內深計算會自動考慮：

- 內崁門占用前方深度
- 內嵌背板占用後方深度
- 主結構板厚

外尺寸反推邏輯：

```text
外寬 = 內寬 + 左右板厚
外高 = 內高 + 上下板厚
外深 = 內深 + 內崁門占用 + 內嵌背板占用
```

### 3) 右側未使用功能自動隱藏

未勾選或未啟用時，右側相關欄位會自動收合：

- 未勾背板 → 隱藏背板板材
- 無門片 → 隱藏門片板材
- 未勾抽屜 → 隱藏抽屜側板 / 底板板材
- 無門片且無抽屜 → 隱藏五金估算
- 箱體模式 → 隱藏櫃體專用板材與五金

### 4) 手機版維持易讀

保留手機版卡片式裁切清單與不需放大縮小的版面。
